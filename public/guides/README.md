# PDF Guides

Place your PDF files here:

- `enso-breathwork-guide.pdf` - Universal guide (single PDF for all results)

**OR** create personalized versions:

- `enso-calm-seeker.pdf` - For calm result type
- `enso-focus-builder.pdf` - For focus result type
- `enso-rest-restorer.pdf` - For sleep result type
- `enso-energy-cultivator.pdf` - For energy result type

## Creating the PDF

Use the content in `pdf-content.md` (same folder) to design in Canva, Figma, or similar.

Recommended Canva template size: **US Letter (8.5 x 11 in)**

## Quick Canva Workflow

1. Go to canva.com → Create design → Document (US Letter)
2. Copy content from `pdf-content.md`
3. Use brand colors:
   - Background: #FAFAF9
   - Text: #292524
   - Accent: #B45309 (amber)
   - Slate: #64748B
4. Export as PDF and place here
